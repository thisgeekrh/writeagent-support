About WriteAgent

WriteAgent is a production-ready iOS app designed specifically for real estate agents to generate high-quality marketing content using AI. With a modern "liquid glass" aesthetic and secure API integration, it helps you create engaging content for your real estate business.

Key Features

  • AI-powered content generation using OpenAI's API
  • Secure API key storage with iOS Keychain
  • Modern liquid glass UI design
  • Tailored for real estate professionals
  • Multiple content types and formats
  • Profile customization with service areas
  • Daily social media inspiration

Getting Started

1. Set Up Your API Key

To use WriteAgent, you'll need an OpenAI API key:

  • Navigate to the Profile tab in the app
  • Tap "Edit Profile"
  • Enter your OpenAI API key in the "ChatGPT API Key" field
  • Tap "Save API Key"

2. Configure Your Profile

Set up your professional details for personalized content:

  • Your Name
  • Business/Team Name
  • Service Areas (cities/states you serve)
  • Content Tone preference

3. Generate Content

Use the Create tab or Quick Actions from the Home tab to generate content. Add specific details or context, then tap "Generate Content" to create your marketing materials.

Privacy Policy

Last Updated: December 2024

Information We Collect

WriteAgent is designed with privacy in mind. We collect minimal information:

  • No Personal Data Collection: We do not collect, store, or transmit any personal information
  • Local Storage Only: Your profile information (name, business name, service areas) is stored locally on your device
  • API Keys: Your OpenAI API key is stored securely in iOS Keychain and never transmitted to our servers
  • Generated Content: Content you generate is not stored or transmitted to our servers

How We Use Your Information

Since we don't collect personal information, there's nothing to use:

  • No data analytics or tracking
  • No user behavior monitoring
  • No content analysis or storage
  • No third-party data sharing

Data Security

Your data security is our priority:

  • iOS Keychain: API keys are encrypted and stored securely using Apple's Keychain Services
  • Local Processing: All app data remains on your device
  • No Server Storage: We don't operate servers that could store your data
  • OpenAI Integration: Content generation happens directly between your device and OpenAI's secure API

Third-Party Services

WriteAgent integrates with the following third-party services:

  • OpenAI API: For AI content generation. Your API key and generated content are handled according to OpenAI's privacy policy
  • Apple Services: Standard iOS services for app functionality (Keychain, etc.)

Your Rights

You have complete control over your data:

  • Delete Data: Uninstalling the app removes all local data
  • API Key Control: You can change or remove your API key at any time
  • Profile Management: You can modify or clear your profile information anytime
  • No Data Portability Needed: Since we don't collect data, there's nothing to export

Children's Privacy

WriteAgent is not intended for children under 13. We do not knowingly collect information from children under 13.
