Getting Started

🔑 How to Get Your OpenAI API Key

1

Visit OpenAI's Website

Go to platform.openai.com and sign up for an account or log in if you already have one.

2

Navigate to API Keys

Once logged in, click on your profile icon in the top right, then select "API Keys" from the dropdown menu.

3

Create a New API Key

Click "Create new secret key" and give it a name like "WriteAgent". Copy the key immediately - you won't be able to see it again!

4

Add Credits to Your Account

Go to "Billing" and add credits to your account. OpenAI charges per usage, typically a few cents per content generation.

💡 Pro Tip

Start with $5-10 in credits to test the app. Most users spend less than $1 per month for regular content generation.

1. Set Up Your API Key in WriteAgent

Once you have your OpenAI API key:

  • Open WriteAgent and navigate to the Profile tab
  • Tap "Edit Profile"
  • Enter your OpenAI API key in the "ChatGPT API Key" field
  • Tap "Save API Key"

2. Configure Your Profile

Set up your professional details for personalized content:

  • Your Name
  • Business/Team Name
  • Service Areas (cities/states you serve)
  • Content Tone preference

3. Generate Content

Use the Create tab or Quick Actions from the Home tab to generate content. Add specific details or context, then tap "Generate Content" to create your marketing materials.

Troubleshooting

❌ "Could not generate content"

Solution: Check your API key and ensure you have a stable internet connection. Verify your OpenAI API key is valid and has sufficient credits.

🔑 API key not saving

Solution: Ensure you have a valid OpenAI API key starting with "sk-". Check that you have sufficient permissions and credits in your OpenAI account.

💥 App crashes

Solution: Make sure you're running on iOS 17.0 or later. Try restarting the app and your device if issues persist.

🌐 Connection issues

Solution: Ensure you have a stable internet connection. Check your network settings and try switching between WiFi and cellular data.
